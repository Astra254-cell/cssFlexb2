# Flexbox Layout Project

This is my first project using CSS Flexbox to build a responsive webpage layout.

## 🚀 Project Overview
The goal of this project was to practice modern CSS layout techniques using Flexbox and improve my web development skills.

## 🛠 Technologies Used
- HTML5
- CSS3
- Flexbox
- Font Awesome Icons

## ✨ Features
- Responsive layout
- Styled navigation bar
- Flexbox content sections
- Footer with custom caption
- Icons integrated into the design

## 📚 What I Learned
Through this project, I learned how to:
- Use Flexbox for page layout
- Align and space elements properly
- Style a responsive webpage
- Organize CSS files and assets
